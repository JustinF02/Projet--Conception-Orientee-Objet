Bonjour !

Ce README a pour seul but d'expliquer le fonctionnement du programme.

La seule manoeuvre que doit effectuer l'utilisateur est de mettre le fichier de carte
dans le dossier .\projetGuybrush\carte (en comptant ce dossier comme dossier courant)

Une fois le programme lancé, vous aurez la possibilité de rentrer un nom de fichier, il ne faut pas oublier l'extension !
Il faudra par exemple rentrer scabb.clair ou phatt.chiffre. Le programme détectera quel type de carte il s'agit et rendra disponible certaines actions
en conséquence.

Les cartes scabb et phatt sont toutes deux disponibles en version claire et en version chiffrée, courtoisie de l'équipe CSI !
(à noter que les fichiers scabb.chiffre et phatt.chiffre ont été réalisés avec l'algorithme de codage du projet !)
